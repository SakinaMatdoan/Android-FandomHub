# Proyek Aplikasi Android: Fandom Hub

Fandom Hub adalah sebuah aplikasi mobile yang dikembangkan sebagai platform komunitas untuk para penggemar (fans). Aplikasi ini bertujuan untuk menjadi wadah di mana pengguna dengan minat yang sama dapat terhubung, berdiskusi, dan berbagi konten.

Proyek ini merupakan hasil tugas untuk memenuhi penilaian mata kuliah Pemrograman Mobile.

## Deskripsi Aplikasi

Aplikasi Fandom Hub dirancang untuk mengatasi kebutuhan para penggemar yang seringkali tersebar di berbagai platform media sosial. Dengan Fandom Hub, pengguna dapat bergabung ke dalam "Hub" (ruang komunitas) yang spesifik sesuai dengan minat mereka, seperti grup musik, film, serial TV, atau game.

## Fitur Utama

-   **Autentikasi Pengguna**: Pengguna dapat membuat akun dan masuk ke dalam aplikasi secara aman.
-   **Profil Pengguna**: Setiap pengguna memiliki halaman profil yang dapat dikelola, termasuk mengganti foto profil.
-   **Navigasi Intuitif**: Antarmuka yang mudah digunakan untuk berpindah antar halaman, seperti halaman "Akun", "Marketplace", dan lainnya.
-   **Penyimpanan Data Lokal**: Aplikasi memanfaatkan database lokal untuk menyimpan data penting, memungkinkan akses yang cepat dan pengalaman pengguna yang lebih baik.

## Teknologi yang Digunakan

Aplikasi ini dibangun menggunakan tumpukan teknologi modern untuk pengembangan Android:

-   **Bahasa Pemrograman**: Kotlin
-   **Antarmuka Pengguna (UI)**: Jetpack Compose (sebuah toolkit modern untuk membangun UI native Android secara deklaratif).
-   **Arsitektur**: Mengikuti prinsip-prinsip arsitektur yang bersih dan modular (seperti MVVM).
-   **Navigasi**: Menggunakan Navigation Compose untuk mengatur alur perpindahan antar layar.
-   **Database Lokal**: Room Persistence Library untuk manajemen database SQLite.
-   **Penyimpanan Data Sederhana**: DataStore untuk menyimpan preferensi pengguna seperti status login.
-   **Pemuatan Gambar**: Library Coil untuk memuat gambar dari internet atau lokal secara efisien.
